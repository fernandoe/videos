[Django, Travis, Deploy, Dreamhost] Automatizando o deploy do projeto com Travis para a Dreamhost


DESCRIÇÃO:
Nesse vídeo fiz a configuração do projeto no Travis para o deploy automático da sua última versão quando um novo commit for identificado. Vale lembrar que o deploy só acontece se todos os testes da aplicação executarem com sucesso. O deploy vai para um server dentro da Dreamhost.
