TÍTULO:
[Python, Django, Deploy, Dreamhost, Git, Pyenv] Automatizando o deploy com git push (push-to-deploy)


DESCRIÇÃO:
Nesse vídeo fizemos a configuração do deploy usando git push (push-to-deploy) para um servidor dentro da Dreamhost. Fizemos todo o processo do zero, configurações no servidor e no projeto, instalações de dependências e ferramentas para o gerenciamento de virtualenvs. No final deixamos a aplicação funcionando dentro do domínio: https://arquivo.fe.fernandoe.com
